# CampusHire Frontend (React + TypeScript + Vite)

## Prerequisites
- Node.js 18+
- npm (bundled with Node)

## Setup
```bash
cd frontend
npm install
```

## Development
```bash
npm run dev
```
Open the printed URL (default http://localhost:5173).

## Build
```bash
npm run build
```
Outputs production build to `dist/`.

## Preview built app
```bash
npm run preview
```
